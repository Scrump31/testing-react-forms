## Create a .env file

```
MONGODB_URI="database creds here"
```

## Installing/Running the app

```bash
npm install
npm run dev
```

## Running Tests

```bash
npm test
npm run cy:open
```

### Formatting with Prettier

```bash
npm run format
```
